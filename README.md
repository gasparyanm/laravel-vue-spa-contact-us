## Features

- Laravel 8
- Vue + VueRouter + Vuex + VueAxios

## Installation

- `git clone https://github.com/gasparyanm/laravel-vue-spa-contact-us.git`
-  Edit `.env` and set your database connection and email config details
- `php artisan key:generate`
- `php artisan migrate`
- `npm install`

## Usage

#### Development

```bash
npm run dev
```
